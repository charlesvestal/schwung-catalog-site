What is Schwung?

Schwung (formerly Move Everything) is an unofficial framework for the Ableton Move. It adds a Shadow UI that runs in parallel with stock Move, letting you layer additional synths, effects, and overtake apps on top of the device you already use. Move keeps working normally — Schwung just opens it up.

It's MIT-licensed, built on top of the Move Anything project, and developed in public on GitHub.

What Schwung adds

Platform-level capabilities that ship with the host — no separate install.

Shadow UI

Runs custom signal chains alongside stock Move so additional synths and effects layer in without leaving the device.

Skipback

Shift+Capture writes the last 30 seconds of audio to disk — recover happy accidents you weren't recording.

Quantized Resampler

Bar-locked resampling of Schwung synths or Move input. Recording starts on a note event or play.

Song Mode

Arrange clips from your set into ordered steps that advance automatically by bar timing across all 4 tracks.

RNBO Runner

Host RNBO graphs as an overtake module with full pad/knob control and per-set state.

Schwung Manager

Web UI at move.local:7700 for managing modules, files, settings, and updates from any browser.

Screen Reader

Optional TTS announcements for accessibility — toggle from Shadow UI settings or Shift+Menu.

Native Sampler Bridge

Route Schwung's master output into Move's native sampling workflow as a Line In source.

Module ecosystem

Community-built synths, effects, and tools you install on top of Schwung.
