What is Schwung?

Schwung is an unofficial framework for the Ableton Move. It adds a parallel ("Shadow") UI that runs alongside stock Move, letting you layer additional synths and effects and run completely new tools on top of the device you already use. Move keeps working normally — Schwung just opens it up.

At a glance:

  • Emulations of classic synthesizers, and entirely new possibilities like granular synthesis and advanced soundfont support.
  • New sampling sources like AirPlay, YouTube and SoundCloud streaming, or the serendipitous Crate Dig.
  • Audio effects like tape delay, luscious reverbs, vocoders, and tape distortion.
  • Tools like an on-device waveform editor, timestretching, and stem separation.
  • Workflow enhancements like Song Mode, always-running skipback recording, and a quantized resampler.
  • All managed from an easy-to-use web interface.

What Schwung adds

Platform-level capabilities that ship with the host — no separate install.

Shadow UI

Runs custom signal chains alongside stock Move so additional synths and effects layer in without leaving the device.

Song Mode

Arrange clips from your set into ordered steps that advance automatically by bar timing across all 4 tracks.

Skipback

Shift+Capture writes the last 30 seconds of audio to disk — recover happy accidents you weren't recording.

Quantized Resampler

Bar-locked resampling of Move audio to make perfect loops.

Screen Mirroring

View your Move's screen in a web browser for screencasting or giving your eyes a break.

RNBO Runner

Host RNBO graphs as an overtake module running alongside your Move set.

Schwung Manager

Web UI at move.local:7700 for managing modules, files, settings, and updates from any browser.

Screen Reader

Optional TTS announcements for accessibility making Move the first musical instrument with an on-device screen reader..
